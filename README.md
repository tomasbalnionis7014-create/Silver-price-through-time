# Silver Price Through Time
this project shows the historical prices of silver. this was constructed from the data set that is attached.
- 7 day and 14 day moving averages calculated
- plots price
- includes PNG visualisation

- ## files
-  `silver_price_through_time.py` — main script
- `Silver Futures Historical Data.csv` — data
- `silver_price_through_time.png` — plot
